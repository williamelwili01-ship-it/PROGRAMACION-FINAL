Nombre del proyecto 
juego de piedra papel o tijera

William Guatapi

Objetivo del sistema 
El objetivo es aplicar los conocimientos adquiridos durante la asignatura para crear un programa interactivo que permita a un usuario jugar contra la computadora.


Descripción de funcionalidades 
el jugar 1 debe elegir entre piedra, papel o tijera de la misma manera el jugador 2 
de esta forma el sistema  del programa dara el resultado dependiendo de las opciones elegidas de los 2 jugadores
